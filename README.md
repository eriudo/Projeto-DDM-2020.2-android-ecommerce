# Projeto para a disciplina de Desenvolvimento para Dispositivos Móveis 2020.2

## Sistema de Vendas Online

O projeto tem como finalidade a nota da terceira unidade da disciplina Desenvolvimento para Dispositvos Móveis, o sistema consiste em um aplicativo de vendas online, programado na linguagem Kotlin.

A comunicação com o banco de dados é a partir do Firebase, e o app consome uma API, para aplicar as regras de negócio.
